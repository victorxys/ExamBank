# Product Overview

ExamBank (萌姨萌嫂企业管理系统) is an enterprise-grade management platform combining exam/training systems with comprehensive HR and financial management.

## Core Modules

**Exam & Training System**
- Question bank management with multiple question types (single/multiple choice, Q&A)
- Automated exam paper generation and grading
- Knowledge point tracking and analysis
- Course resource management with media playback tracking
- TTS audio generation and video synthesis

**Employee Evaluation System**
- Multi-level evaluation structure (Aspects → Categories → Items)
- Support for internal, customer, and self-evaluations
- AI-powered evaluation summaries using LLM integration

**Financial Management**
- Contract lifecycle management (multiple contract types: childcare, maternity, trial, substitute)
- Automated billing for customers and employee payrolls
- Bank statement reconciliation with intelligent matching
- Financial adjustments and payment tracking
- Monthly settlement reports

**User & Permission Management**
- Role-based access control (Admin/Teacher/Student)
- JWT authentication with 30-day token expiration
- Course and resource-level permissions
- External API authentication for AI chatbot integration

## Target Users

- Internal staff (administrators, teachers, HR personnel)
- Service workers (maternity nurses, childcare workers)
- Customers (service recipients)
- External systems (AI chatbot via API key authentication)
