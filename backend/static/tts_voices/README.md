# TTS 参考音频目录

此目录用于存放 IndexTTS2 的参考音频文件。

## 使用说明

1. 将参考音频文件（WAV 格式推荐）放置在此目录下
2. 在 TTS 配置中填写文件名（如 `default_voice.wav`）
3. IndexTTS2 将使用该音频作为音色克隆的参考

## 音频要求

- 格式：WAV、MP3 等常见音频格式
- 时长：建议 5-30 秒
- 质量：清晰、无噪音的语音录音
- 内容：最好是目标说话人的自然语音

## 示例文件

- `default_voice.wav` - 默认参考音频（需要自行添加）
- `emotion_happy.wav` - 开心情感参考（可选）
- `emotion_calm.wav` - 平静情感参考（可选）
